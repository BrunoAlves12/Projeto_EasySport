# EasySport

EasySport é uma aplicação web desenvolvida em Flask para gestão de reservas de espaços desportivos.
Permite o registo e autenticação de utilizadores, criação de reservas e gestão por parte do administrador.

## Objetivo

O objetivo do EasySport é facilitar a gestão de espaços desportivos e respetivas reservas,
permitindo aos utilizadores reservar horários disponíveis e ao administrador controlar os espaços,
utilizadores e estados das reservas.
